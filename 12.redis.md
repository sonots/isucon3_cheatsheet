# redis のインストール

epel から入れる

```
$ sudo yum install redis
```

入ったファイル。/etc/redis.conf が設定ファイル. ログは /var/log/redis/redis.log に出る
```
$ sudo rpm -ql redis
/etc/logrotate.d/redis
/etc/rc.d/init.d/redis
/etc/redis.conf
/usr/bin/redis-benchmark
/usr/bin/redis-check-aof

/usr/sbin/redis-server
```
